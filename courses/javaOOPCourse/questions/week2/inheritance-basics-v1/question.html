Task: Write a Java program based on the requirements below:

  1. Create a base class Animal with a private field String name and a constructor that accepts the name
  2. Add a public method getName() that returns the name
  3. Create a derived class Dog that extends Animal with a constructor that accepts a name and passes it to the parent constructor
  4. In the Dog class, add a method bark() that returns the string "Woof! My name is [name]"
  5. In the main method, create a Dog object with name "Buddy" and call both getName() and bark() methods

Example: When you run the program, it should output:

Name: Buddy
Woof! My name is Buddy
